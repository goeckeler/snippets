# 2. Plain C

Date: 2026-03-17

## Status

Accepted

## Context

It is an assignment to showcase C programming skills.

## Decision

The code must be written in plain ANSI-C, not in any other language. C++ is explicitely discouraged. The code must be compilable from the command line with the source code only.

## Consequences

You can use any tools, editors, IDEs, and so on as of your liking, but the code needs to be compiled by a standard C compiler, e.g. gcc.
